# Vagrant

Setup
 
```
$ mkdir ubuntu
$ cd ubuntu
$ vagrant init ubuntu/xenial64
$ vagrant up

$ vagrant ssh-config
$ vagrant ssh
```

Open Linux Box on virtual machine

```
$ cd
$ cd ubuntu
$ vagrant up
$ vagrant ssh
```
